Cumo Unit Tests

Testpyramide

+ Manuelle Tests

Unit tests

  • Testen die individuelle Units im Code
  • Unterstützen den Agile Mindset
  • Änderungen im Code sind einfach zu machen
  • Bugs werden sofort entdeckt
  • Sind super einfach und schnell
  • Test Frameworks: Jasmine, Jest, Mocha

Service (Integration, E2E) Tests

  • Testen die API (REST, Komponent ...)
  • Testen die Verbindung zwischen der App und der API
  • Testen die Integration von App Komponenten

UI (Funktionalle) Tests

  • Testen die App von der Nutzer Perspektive
  • Testen die Flows (Screen- und Aktionreihenfolge)
